Fungsi:
Untuk menghitung biaya total pengiriman berdasarkan berat, jarak, dan volume paket.

Cara kerja:

Pengguna memasukkan berat, jarak, panjang, lebar, dan tinggi.

Program menghitung volume.

Jika jarak ≤ 10 km → biaya Rp 4250/kg, kalau lebih → Rp 6000/kg.

Kalau volume lebih dari 100 cm³, ada tambahan Rp 50.000.

Hasil akhirnya ditampilkan ke layar.
